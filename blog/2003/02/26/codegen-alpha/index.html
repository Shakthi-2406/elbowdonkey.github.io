codeGen Alpha

The code generation tool I mentioned yesterday is now available

. It is still a work in progress, and hammering away at it will help work out the kinks. I’ve already used it on a couple of things, and even with the quirks, it’s still a huge time saver.


You’ll get the most out of it if you can consume the component within your own app. There’s no documentation yet, but that will come as the tool becomes more refined.


Let me know about any bugs.
